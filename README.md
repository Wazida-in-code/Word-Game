# Word Guessing Game 🎮

A simple and fun word guessing game built with Python. Unscramble jumbled words and test your guessing skills!

## How to Play

1. **Run the game:**
   ```bash
   python word_game.py
   ```

2. **Enter your name** when prompted

3. **Guess the word** from the jumbled letters
   - You have **5 chances** to guess each word
   - Each correct guess gives you **10 points**
   - Try to get the highest score!

4. **Keep playing** or quit after each round

## Features

✨ **Colorful Console Output** - Beautiful ANSI colored text for better gameplay experience

🎯 **Scoring System** - Earn 10 points for every correct guess

🔄 **Multiple Rounds** - Play as many rounds as you want

🎲 **Random Words** - Randomly selected words from a word list

⏱️ **Loading Animation** - Cool loading screen at startup

## Requirements
